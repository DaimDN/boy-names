
# Boys name generator 


Generate Random names for baby Boys. Use names.boysname() function and its going to generate a random name for baby boy.


## Install

```
$ npm i boy-names
```


## Usage

```
const names = require("boys-name");

var name = names.boysname();

console.log(name);

```
